# Changelog

All notable changes to RiskLink Lite should be documented in this file.

---

## v1.0.0

Initial public documentation release.

### Added

- Modular account risk scoring framework
- AccountAge signal
- JoinBurst signal
- Behavior signal
- NamePattern signal
- Avatar placeholder signal
- Trust profile system
- Recommendation system
- Safelist support
- Risk tags
- Display summary output
- Player attributes
- RiskChanged event hook
- Manual reevaluation API
- Test utilities
- In-memory moderation queue support
- Moderation feed events
- Review queue
- High-risk queue
- Moderation snapshot support
- Policy foundation
- Webhook-ready placeholder support
- Documentation structure

### Notes

- Lite version is intentionally conservative
- Auto moderation is disabled by default
- Avatar scoring is disabled by default in Lite
- Webhook support is placeholder/foundation-only in Lite
