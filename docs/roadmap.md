# Roadmap

This document outlines the planned direction of RiskLink Lite and future RiskLink Pro development.

---

## RiskLink Lite Goals

RiskLink Lite is intended to remain:

- lightweight
- easy to install
- easy to understand
- safe by default
- practical for moderation workflows
- modular and expandable

---

## Completed in Lite

- Modular signal architecture
- Config-driven risk labels
- Recommendation system
- Trust profile system
- Safelist support
- Summary output
- Risk tags
- RiskChanged event
- Manual reevaluation
- Player attributes
- Test utilities
- Moderation feed foundation
- In-memory flagged player queue
- Review queue
- High-risk queue
- Moderation snapshots
- Policy foundation
- Webhook-ready placeholder structure

---

## RiskLink Pro Direction

Possible Pro features include:

- stronger behavioral correlation
- cross-session trust history
- moderation note integration
- punishment-linked risk scoring
- richer webhook / external logging support
- advanced join-wave correlation
- admin system integration
- moderation adapters
- Ban API adapter support
- richer analytics
- stronger signal set
- more configurable moderation policies
